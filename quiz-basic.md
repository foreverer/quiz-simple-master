# 前端开发的概念

## Q1

名词解释，前端开发的语境中，这些东西都是什么？

越精简越好，用一两句话你表现出的理解，例如：`Node.js 是 JS 的一个 runtime`  
（不了解的可以跳过）

请在下面的表格中作答：

| 名词           | 是什么                              |
| :------------- | :---------------------------------- |
| Node.js        | 是 JS 的一个 runtime                |
| npm            | 是 Node.js的一个打包工具            |
| npx            | 是 基于npm的一个工具                |
| nvm            | 是 node切换版本的一个工具           |
| nrm            | 是 npm的一个镜像管理工具            |
| yarn           | 是 Hadoop的一个资源管理器           |

| Electron       | 是 一个跨平台的桌面应用开发工具     |
